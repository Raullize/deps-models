## Descrição

Landing page e portfólio da **DEPS Models**, uma agência especializada no desenvolvimento de sistemas sob medida, plataformas de e-commerce e landing pages de alta conversão. O projeto conta com design moderno, animações avançadas e performance otimizada focada na captação de clientes.

## Tecnologias Utilizadas

- **Next.js** (v16)
- **React** (v19)
- **Tailwind CSS** (v4)
- **TypeScript**
- **Framer Motion & GSAP** (Animações complexas e micro-interações)
- **Three.js & OGL** (Efeitos visuais em Canvas/3D)
- **Lenis** (Smooth Scrolling)

## Como Executar o Projeto

### Pré-requisitos

- Node.js (v20 ou superior)
- pnpm (Gerenciador de pacotes)

### Instalação

1. **Clone o repositório**
   ```bash
   git clone https://github.com/depsModels/deps-models
   cd deps-models
   ```

2. **Instale as dependências**
   ```bash
   pnpm install
   ```

3. **Inicie o servidor de desenvolvimento**
   ```bash
   pnpm dev
   ```

   A aplicação estará disponível em `http://localhost:3000`.
